# Ejercicios de Java

**Autor:** Emmanuel Antonio Dzib Galeana  

## Descripción
Este repositorio contiene una colección de ejercicios desarrollados en Java como parte de prácticas académicas.  
Los ejercicios están organizados en diferentes carpetas, cada una representando un ejercicio independiente, con el objetivo de aplicar conceptos fundamentales de programación.

---

## Contenido

###  Ejercicio 1
Programa básico que muestra mensajes en consola y refuerza la estructura de un programa en Java.

###  Ejercicio 2
Implementación de una calculadora sencilla utilizando operaciones básicas, haciendo uso de variables y operadores.

###  Ejercicio 3
Clasificación de números mediante estructuras de control (condicionales), permitiendo determinar diferentes casos según el valor ingresado.

###  Ejercicio 4
Uso de clases y objetos mediante la creación de una clase `Estudiante`, aplicando conceptos de Programación Orientada a Objetos (POO).

###  Ejercicio 5
Ejemplo de herencia, interfaces y polimorfismo mediante una jerarquía de vehículos (`Vehiculo`, `Auto`, `Moto`, etc.).

###  Ejercicio 6
Manipulación de cadenas de texto utilizando métodos de la clase `String`.

---

## Temas aplicados

- Variables y tipos de datos
- Estructuras de control (if, else, etc.)
- Programación Orientada a Objetos (POO)
  - Clases y objetos
  - Herencia
  - Interfaces
  - Polimorfismo
- Colecciones (ArrayList)
- Manejo de Strings
